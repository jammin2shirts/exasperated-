# Moving Assistant App

A web application to help users make informed moving decisions across US states.

## Features

- **Salary & Tax Calculator** – See how your salary translates after state income taxes when moving to a new state
- **Best Places to Live** – Ranked recommendations by employment opportunities and cost of living/affordability
- **K-12 School Comparison** – Compare school system quality between your current state and potential destinations

## Tech Stack

- [Next.js](https://nextjs.org) (App Router)
- [TypeScript](https://www.typescriptlang.org)
- [Tailwind CSS](https://tailwindcss.com)

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run start` | Start production server |
| `npm run lint` | Run ESLint |
